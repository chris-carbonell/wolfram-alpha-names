# Overview

**wolfram-alpha-names** summarizes the info in the **Wolfram Alpha** API into an email that I receive daily to help inspire names for the **baby**

# Prerequisites

1. Operating System\
   This code has only been tested on Ubuntu 20.04.1 LTS.
2. Infrastructure
   Python 3.8.0
3. Knowledge
   - Docker
   - Jupyter
   - Wolfram Alpha
   - urllib
   - smtplib, email
   - cron
